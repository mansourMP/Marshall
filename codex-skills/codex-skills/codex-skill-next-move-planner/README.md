# next-move-planner

A Codex skill for deciding the highest-leverage next step and drafting the next clean prompt, handoff, or execution request.

## What it provides
- A practical prioritization workflow in `SKILL.md`
- Heuristics for choosing the best next move
- Prompt shapes for turning messy context into one actionable request

## Good fit for
- product cleanup and prioritization
- deciding the next Codex task
- reducing ambiguous project state into one action
- drafting the next prompt for yourself or another agent

## Skill contents
- `SKILL.md`
- `agents/openai.yaml`
- `references/priority-heuristics.md`
- `references/next-prompt-shapes.md`
- `references/stop-rules.md`

## Install
Copy the `next-move-planner` folder into `~/.codex/skills/`.

## Example use
`Use $next-move-planner to choose the highest-leverage next step for this project and draft the next prompt for Codex.`
