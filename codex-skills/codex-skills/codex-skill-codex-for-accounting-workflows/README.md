# codex-for-accounting-workflows

A Codex skill for safe, narrow accounting-support workflows. It focuses on triage, summaries, checklist support, and draft-first outputs with explicit approval boundaries.

## What it provides
- A practical accounting-support workflow in `SKILL.md`
- Examples for finance inboxes, statement summaries, and checklist-based work
- Review boundary guidance for what may be drafted, what must be approved, and what should stay blocked

## Good fit for
- accounting managers and controllers
- finance admins and coordinators
- founders doing light finance operations
- mixed-technical teams that need trust-first workflows

## Skill contents
- `SKILL.md`
- `agents/openai.yaml`
- `references/workflow-examples.md`
- `references/review-boundaries.md`
- `references/draft-first-model.md`

## Install
Copy the `codex-for-accounting-workflows` folder into `~/.codex/skills/`.

## Example use
`Use $codex-for-accounting-workflows to triage this finance inbox, summarize the attached statements, and draft the missing-document follow-ups for review.`
