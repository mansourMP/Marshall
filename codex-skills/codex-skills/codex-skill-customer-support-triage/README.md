# customer-support-triage

A Codex skill for turning messy support queues into structured priorities, draft replies, and escalation-ready handoff notes.

## What it provides
- A practical support triage workflow in `SKILL.md`
- Queue, escalation, and reply references for daily support work
- A trust-first model that avoids unsupported customer promises

## Good fit for
- support leads
- ops teams handling support overflow
- founders handling early customer support
- mixed support and product teams

## Skill contents
- `SKILL.md`
- `agents/openai.yaml`
- `references/queue-patterns.md`
- `references/escalation-boundaries.md`
- `references/reply-patterns.md`

## Install
Copy the `customer-support-triage` folder into `~/.codex/skills/`.

## Example use
`Use $customer-support-triage to sort these customer tickets by urgency, prepare the escalations, and draft replies for review.`
